# klstudios-site

## Content management

The main copy, calls to action, links, and image references now live in [content-template.js](content-template.js). Update that file to change the site content without editing the page markup.

### How to use it
- Edit the values under the relevant page in [content-template.js](content-template.js)
- Replace image paths in the same file to swap photos or logos
- Keep the existing page structure; the scripts will render the content into the matching sections
